# DANYA PROAI — лендинг курса по ИИ-контенту

Премиальный одностраничный лендинг (12 секций) по ТЗ Данилы Кима.
Эстетика: нео-оригами × sumi-e × ренессансная фреска.
Палитра: чернила `#0A0A0A` · бумага `#F4F1EA` · салатовый `#C8FF3E`.

## Стек
- **HTML + CSS + JS** в одном файле `index.html` (инлайн-стили и скрипты).
- **Lenis** — плавный скролл (CDN).
- **GSAP + ScrollTrigger** — параллакс героя, scroll-choreography весов / стопки / двери, reveal-ы (CDN).
- Шрифты: Unbounded, Manrope, JetBrains Mono (Google Fonts).
- Картинки лежат в `./assets/` (локально). При желании можно вернуть Cloudinary-ссылки из ТЗ.

## Что внутри
1. Hero — параллакс-фреска «Сотворение Адама» (7 слоёв) + метафора кисти
2. Социальное доказательство — клиенты + метрики (счётчики)
3. Боль — 3 сценария (тёмная секция)
4. Решение — 4 карточки
5. Кому подойдёт / не подойдёт
6. Главное отличие — оригами-весы
7. Программа — 9 уроков / 3 недели + стопка-оригами
8. Формат — сетка 2×3
9. Об авторе — фото + био + кейсы
10. Стоимость — цена + сравнение экономики
11. FAQ — аккордеон
12. Финальный CTA — оригами-дверь + форма заявки

## Форма → Telegram
Заявка уходит `POST`-ом в Telegram Bot API прямо из браузера (как в ТЗ).
**⚠️ Токен бота виден в коде страницы.** Перед публичным запуском вынесите его
на serverless-прокси (Netlify/Vercel function), иначе бота можно заспамить.
Токен/chat_id заданы в начале `<script>` (`TELEGRAM_BOT_TOKEN`, `TELEGRAM_CHAT_ID`).

## Локальный просмотр
```bash
python3 serve.py 5577      # затем открыть http://127.0.0.1:5577
```

## Деплой
Залить `index.html` + папку `assets/` на любой статический хостинг
(Netlify, GitHub Pages, Vercel, Tilda-embed). Внешний бэкенд не нужен.

## Доступность / адаптив
- Mobile-first, проверено на 375 / 768 / 1440 px.
- Параллакс отключается на узких экранах и при `prefers-reduced-motion`.
- Touch-таргеты ≥ 44px, фокус-стейты, semantic-инпуты.
